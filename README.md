# 🧠 Handwritten Digit Recognizer

A deep learning project developed during my Semester 4 Internship at Codec Technologies.

## 📌 Project Overview

This project recognizes handwritten digits (0–9) using a Convolutional Neural Network (CNN) trained on the MNIST dataset.

## 🚀 Features

- CNN-based digit recognition
- Trained on the MNIST dataset
- Around 99% test accuracy
- Image upload and prediction
- Confidence score
- Training & validation graphs
- Confusion matrix
- Classification report

## 🛠️ Technologies Used

- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- OpenCV
- Google Colab

## 📂 Files

- `Handwritten_Digit_Recognizer.ipynb` – Complete notebook
- `digit_recognizer.keras` – Trained model
